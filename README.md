# 💫 About Me
Hi, I’m Shaheer 👋 

I’m a final-year BSc Mathematics and Computer Science student. I spend most of my time trying to find the sweet spot between hard logic and visual art. 

When I’m not writing code or studying algorithms, I’m usually working on 2D/3D art and animation, playing the drums, or flying planes (I have my private pilot's license!). 

I learn best by actually getting my hands dirty and building things—whether that means writing a game loop, sculpting a 3D model, or figuring out a new animation technique. 

Always exploring, always building.
